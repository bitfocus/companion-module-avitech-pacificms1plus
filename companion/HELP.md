## Avitech Pacific MS Series (1) Control Module

This is a module for the Avitech Pacific MS Series 1

## Getting started

This module contains commands to control the Avitech Pacific MS-1 Plus Multiview KVM.

Set your IP address on your Connection config to the same IP Address your Pacific MS-1 Plus is running on in order to send commands. Attach any of the commands for Salvo/Layout/Info/IP-Setting/Machine-Name/or Route setting to a button with the desired input/output ports and slot locations (if needed) and you can send HTTP commands directly to the Pacific MS-1 Plus from your deck. Remember when using strings to attach names or labels to follow the information on the tooltips about ommitting certain characters that will not be able to be read by the Pacific MS-1 Plus.
